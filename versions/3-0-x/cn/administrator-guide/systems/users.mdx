---
title: 系统管理员
---

系统管理员可以在 Settings 页管理当前企业版的系统管理员用户，具备以下权限：

- 登录企业版管理后台
- 添加 / 删除 / 禁用团队成员
- 为 Workspace 添加 Owner
- 为 Workspace 添加 新的成员并分配权限

## 管理系统管理员

### 添加系统管理员

企业版后台的创建者默认是第一位系统管理员。任何一位系统管理员都能够邀请其他人成为系统管理员。点击 Settings 页右上角的蓝色 Add 按钮，输入用户姓名和邮箱完成添加。

![](https://assets-docs.dify.ai/2025/05/cb2bad848e40760140e7aa6e25eb2a42.png)

如果某位系统管理员的邮箱未加入 "Members" 列表，那么他并没有创建和使用工作区的权限。如果希望系统管理员也能够创建或加入某个 Workspace，请参考[《在工作区新增成员》](/administrator-guide/members-management)。

### 修改系统管理员信息

选择某位系统管理员，点击右侧的菜单按钮，轻点 Edit，在弹出的 tab 页修改系统管理员的名称或邮箱。

![](https://assets-docs.dify.ai/2025/05/174d265a01cca44c60ae34dc9c0ff4a5.png)

### 重置系统管理员密码

如果某位成员遗失了密码，需要系统管理员手动发送重置密码邮件。点击成员右侧的菜单按钮，轻点 Reset Password。

![](https://assets-docs.dify.ai/2025/05/aa143894091c9e4990c5993163df637a.png)

### 禁用系统管理员

选择某位系统管理员，点击右侧的菜单按钮，轻点 Edit，将成员的 Status 切换为 banned。被禁用的管理员将无法访问企业版后台。

![](https://assets-docs.dify.ai/2025/05/3ee32101ff701db88b8b16e7a6db4587.png)

### 删除管理员

⚠️ 危险操作，请谨慎处理
点击系统管理员右侧的菜单按钮，轻点 Delete。

### 搜索管理员

你可以在搜索框内输入邮箱搜索系统管理员。

![](https://assets-docs.dify.ai/2025/05/487f328e311a0242d7216a4a8ce15ba7.png)

## 安全建议

### 两步身份验证（2FA）

**适用对象：** 仅限系统管理员用户

**功能特点：**
- 额外的安全验证层
- 基于 TOTP（Time-based One-Time Password）
- 支持多种认证器应用
- 强制或可选配置

**配置步骤：**

1. 系统管理员登录管理后台
2. 前往 **系统设置** → **安全设置**
3. 启用两步验证功能
4. 扫描二维码绑定认证器
5. 输入验证码完成绑定

详细配置说明请参考 [两步验证](/zh-cn/administrator-guide/2sv)。
