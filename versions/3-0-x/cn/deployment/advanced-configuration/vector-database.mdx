## 在 Helm Chart 值中查找向量数据库配置

### 显示 Helm Chart 值

```bash
helm show values dify/dify
````


## 在 Helm Chart 值中查找向量数据库配置
这是 Helm chart 值中向量数据库配置的示例。

````yaml
vectorDB:
  useExternal: true
  externalType: "qdrant"
  externalQdrant:
    endpoint: "http://your-qdrant-cluster-url.qdrant.tech/"
    apiKey: "#REPLACE_ME#"
  ... ...
````


## `vectorDB.externalType` 字段的选项
### 企业支持的类型
Dify 企业版官方支持以下向量数据库。

- qdrant（建议）
- weaviate（可选）
- elasticsearch（可选）

### 社区支持的类型
Dify 社区版提供以下向量数据库支持。如果你想使用这些向量数据库，Dify 企业版也支持它们，但它们不是官方支持的。

- milvus
- relyt
- pgvecto-rs
- tencent
- analyticdb
- opensearch

> 有关更多详细信息，请参阅文档：[Dify Vector Type](https://github.com/langgenius/dify/blob/main/api/core/rag/datasource/vdb/vector_type.py)
