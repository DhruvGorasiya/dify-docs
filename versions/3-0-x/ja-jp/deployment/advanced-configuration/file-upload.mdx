---
title: ファイルアップロード
---

## Helmチャートの値を表示
````bash
helm show values dify/dify
````

## Helmチャートの値からファイルアップロード設定を確認する
以下は、Helmチャートの値におけるファイルアップロード設定の例です。

````yaml
api:
  limits:
    # ファイルサイズ制限（単位：MB）
    uploadFileSize: 15
    # バッチアップロード数制限
    uploadFileBatchCount: 5
    # 画像ファイルサイズ制限（単位：MB）
    uploadImageFileSize: 5
  ... ...
````
