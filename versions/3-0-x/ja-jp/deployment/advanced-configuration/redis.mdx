## Helm Chartの設定値を表示する
```bash
helm show values dify/dify
```

これはHelm Chartの設定ファイルにおけるRedis設定の一例です。
```yaml
externalRedis:
  enabled: true
  host: "redis.example"
  port: 6379
  username: ""
  password: "#REPLACE_ME#"
  useSSL: false
```

## RedisのSSL設定について
- SSL対応のRedisを利用するには、`useSSL` 項目を `true` に設定してください。
- `port` 項目については、SSL非対応時は `6379` を、SSL対応時は `6380` を指定します。
