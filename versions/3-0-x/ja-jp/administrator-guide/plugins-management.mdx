## プラグインインストールの管理

エンタープライズ版の管理バックエンドで、左側のメニューから **Plugins** をクリックしてプラグイン管理ページにアクセスします。このページでは以下のことができます：

- 現在のプラグインポリシー設定を確認
- インストールを許可するプラグインの種類を制御
- プラグインのインストール元を制限

## 特定のプラグインタイプを許可

管理者は "Allowed Plugins" ドロップダウンメニューを使用して、インストールを許可するプラグインの種類を選択できます：

| プラグインタイプ | 説明 |
|---------|------|
| All | すべての種類のプラグインをインストール可能。公式プラグイン、パートナープラグイン、サードパーティ開発のプラグインを含む |
| Official + Partner | Dify 公式プラグインと認定パートナープラグインのみインストール可能。安全性と安定性のバランスを提供 |
| Official | Dify 公式が開発および維持するプラグインのみインストール可能。高レベルの安全性と安定性を保証 |
| None | いかなるプラグインもインストール禁止。安全性が非常に高い環境に適用 |

**プラグインの出所説明：**

プラグインマーケットでは、タグと作者名を通じて異なる種類のプラグインを識別できます：

## プラグインインストール元の制御

### マーケットインストール制限

"Restrict to Marketplace Installations" スイッチはプラグインのインストール元を制御し、管理者が機能要件と安全リスクをバランスさせるのに役立ちます：

| 設定状態 | 許可されるインストールチャネル | 安全レベル |
|---------|-------------|--------|
| 制限を有効化 | Dify 公式プラグインマーケットのみ | 高 |
| 制限を無効化 | プラグインマーケット、ローカルファイル、GitHub など | 低 |

**プラグインインストール元チャネル一覧：**

| インストール方法 | 安全審査 | バージョン管理 | 使用ドキュメント |
|---------|---------|---------|--------|
| 公式プラグインマーケット | 安全審査済み | 自動バージョン管理と更新 | 詳細な使用ドキュメント |
| ローカルファイルインストール | 自己評価が必要 | 手動管理が必要 | 開発者に依存 |
| GitHub インストール | コードの安全性を確認する必要あり | バージョン更新は迅速だが手動が必要 | プロジェクトのメンテナンスに依存 |

### 署名インストール制限

管理バックエンドでプラグインをマーケットからのみインストールするよう制限する以外に、デプロイ環境ファイルの変数値を変更してプラグインインストールを制御することもできます。

プラグイン署名はプラグインの信頼できる出所の検証と見なされ、プラグインのインストール時に検証が行われます。エンタープライズ版をデプロイする際、環境変数ファイルの `FORCE_VERIFYING_SIGNATURE` はプラグイン署名の検証を制御します。環境変数でプラグイン署名制限を有効にすると、プラグインのインストールに影響します。

- デプロイ時に環境変数を使用して署名インストール制限を制御する方法については、デプロイマニュアルを参照してください。


```yaml
FORCE_VERIFYING_SIGNATURE=false # 強制署名検証を無効化
FORCE_VERIFYING_SIGNATURE=true # 強制署名検証を有効化
```

**署名検証と Marketplace インストール制限の関係一覧：**

### Marketplace インストール制限を有効化

| 環境変数設定 | Marketplace インストール | GitHub オンラインインストール（署名あり） | GitHub オンラインインストール（署名なし） | 手動インストール（署名あり） | 手動インストール（署名なし） |
|------------|-----------------|----------------------|----------------------|------------------|----------------|
| 署名を検証 | ✅ インストール許可 | ❌ インストール不可 | ❌ インストール不可 | ❌ インストール不可 | ❌ インストール不可 |
| 署名を検証しない | ✅ インストール許可 | ❌ インストール不可 | ❌ インストール不可 | ❌ インストール不可 | ❌ インストール不可 |

### Marketplace インストール制限を無効化

| 環境変数設定 | Marketplace インストール | GitHub オンラインインストール（署名あり） | GitHub オンラインインストール（署名なし） | 手動インストール（署名あり） | 手動インストール（署名なし） |
|------------|-----------------|----------------------|----------------------|------------------|----------------|
| 署名を検証 | ✅ インストール許可 | ✅ インストール許可 | ❌ インストール不可 | ✅ インストール許可 | ❌ インストール不可 |
| 署名を検証しない | ✅ インストール許可 | ✅ インストール許可 | ✅ インストール許可 | ✅ インストール許可 | ✅ インストール許可 |

> **備考**：Dify Marketplace で提供されるプラグインはすべて Dify 署名を含んでいます。
