<Warning>
你需要先初始化 Dify Console `http://console.dify.local`，然后才能登录 Dify Dashboard `http://enterprise.dify.local`。
</Warning>

## 1. 初始化 Dify Console
- 访问 `http://console.dify.local`。
- 初始化电子邮件和密码。
- 初始化完成后，你将被重定向到登录页面，并等待许可证激活。

![](https://assets-docs.dify.ai/2025/05/dd983865d955c9c9390aa93b19103419.png)

## 2. 初始化 Dify  Dashboard 以激活许可证
- 访问 `http://enterprise.dify.local`。
- 使用默认的电子邮件和密码登录。你可以在登录后更改密码。
    - 电子邮件： `dashboard@dify.ai`
    - 密码： `difyai123456`
- 许可证激活：登录后，将弹出对话框以激活许可证。
    > 有关更多信息，请参阅 [许可证激活](/versions/3-0-x/zh-cn/deployment/license-activation)。
- 激活许可证后，访问 `http://enterprise.dify.local/dashboard/workspaces` 并创建一个工作空间。

## 3. 从市场安装 `模型` 插件
- 3.1 访问市场。
    - 访问：`http://console.dify.local/plugins`
    - 切换到 `探索市场` 选项卡。
- 3.2 安装模型插件。
    - 从市场中选择一个模型插件。
    - 等待安装完成。（如果是第一次安装插件，可能需要几分钟。）
    - 填写模型插件的密钥令牌。
    - 验证安装成功。

## 3. 验证 Dify Console
- 访问 `http://console.dify.local`。
- 创建一个 `知识库` 并上传一个文件。
- 创建一个 `聊天机器人`，并选择你在 `知识库` 中创建的 `上下文`。
- 选择你从市场安装的模型插件。
- 向聊天机器人提问并验证响应。
