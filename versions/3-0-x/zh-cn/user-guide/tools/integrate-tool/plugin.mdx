[插件](/zh-hans/plugins/introduction)是 Dify 平台内主要的工具类型，同时也是一个面向开发者的开放生态系统。包含五种类型插件，每一种类型插件对应成熟的场景解决方案，赋予开发者用无限的创意改造 Dify 应用的空间。

<Tip>
  推荐使用插件作为首选的工具安装方式。
</Tip>

## 插件与 Dify 内应用的关系

在 Dify 生态系统中，插件和应用是紧密关联的组成部分，它们之间的关系体现在以下几个方面：

### 功能扩展

插件为应用提供了额外的功能和能力：
- **能力增强** - 应用可以通过集成插件获得原本不具备的功能
- **专业化** - 特定领域的插件可以让应用在垂直场景中表现更出色
- **数据连接** - 通过插件，应用可以连接到外部数据源和服务

### 使用流程

1. **插件安装** - 在工作区级别安装并配置插件
2. **应用集成** - 在创建或编辑应用时，从已安装的插件中选择需要的工具
3. **应用调用** - 应用在运行时会调用插件提供的功能

### 权限共享

工作区中安装的插件可以被多个应用共同使用，这意味着：
- 一次授权，多应用受益
- 集中管理插件配置，降低维护成本
- 保持应用间功能的一致性

使用插件作为应用工具具备以下优势：

- **便捷性** - 一键安装，无需复杂配置
- **版本管理** - 插件自带版本管理功能，确保能够应用最新功能
- **集中授权** - 一次授权，工作区全员共享
- **统一体验** - 标准化的工具接口和使用方式

## 安装流程

<Steps>
  <Step title="浏览插件市场">
    前往 [Dify Marketplace](https://marketplace.dify.ai/) 浏览并选择所需要的插件。你可以通过类别、热门程度或关键词搜索找到合适的插件工具。
    
    <Frame>
      ![插件市场界面](https://assets-docs.dify.ai/2025/01/83f9566063db7ae4886f6a139f3f81ff.png)
    </Frame>
  </Step>

  <Step title="安装插件">
    找到所需插件后，点击详情页内右上角的**安装**。
  </Step>

  <Step title="授权配置">
    部分插件可能需要访问外部服务的授权信息。在插件配置页面填写相应的 API 密钥或凭据信息。
    
    <Info>
    所有的授权信息都将被安全加密存储，仅用于与相关服务的通信。
    </Info>
  </Step>

  <Step title="验证授权">
    系统会自动验证你提供的凭据是否有效。验证成功后，插件状态将更新为「已授权」。
    
    <Warning>
    配置凭据后，该工作区中的所有成员都可以在应用程序中使用此工具，但无法查看你输入的实际凭据内容。
    </Warning>
  </Step>

  <Step title="在应用中使用">
    完成安装后，你可以在以下应用类型中使用该插件：
    - **Chatflow** - 会话式流程设计
    - **Workflow** - 自动化工作流
    - **Agent** - 智能代理应用
    
    在这些应用的编辑界面中，选择「工具」→「插件」来启用相应服务。
  </Step>
</Steps>
