### 定義

チャットフローアプリのプロセス内で返信内容を定義します。

テキストエディタでは、返信の形式を柔軟に設定できます。 これには、固定のテキストブロックの作成、前のステップの出力変数を返信内容として利用すること、あるいはカスタムテキストと変数を組み合わせて応答を作成することなどが含まれます。

回答ノードは、対話の応答にコンテンツを動的に組み込むため、どの時点にでもスムーズに連携させることが可能です。 この設定はライブ編集モードに対応しており、テキストと画像を自由に配置できます。例えば：

1. LLMノードの返信内容を出力
2. 生成された画像を出力
3. 純テキストを出力

<Info>
直接返信ノードは最終出力ノードとして使用することもでき、プロセスの中間ステップでストリーミング出力結果を出力することができます。
</Info>
