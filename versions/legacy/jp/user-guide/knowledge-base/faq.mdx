## 1. ドキュメントのインポートやクエリ応答が遅い場合の対処方法は？

通常、ドキュメントをアップロードすると、埋め込み処理に多くのリソースが必要になり、これが遅延の原因となることがあります。サーバーの負荷を確認し、ログをデバッグモードに切り替えたり、埋め込み処理の応答時間を確認してみてください。

## 2. ナレッジベースにアップロードしたドキュメントのコンテンツが多く、セグメント化された内容に異常がある場合の対処方法は？

サーバーの負荷によるメモリ使用状況を確認し、メモリリークが発生していないかをチェックしてください。

## 3. ナレッジベースのファイル処理が「キューイング中」と表示され続ける場合の対処方法は？

この問題は、Redisサービスとの接続が切れてしまい、タスクが完了できないことが原因かもしれません。Workerノードの再起動をお勧めします。

## 4. アプリケーションがナレッジベースのコンテンツ検索結果を最適化する方法は？

異なるパラメータを比較し、検索設定を調整することで、検索結果を最適化できます。詳細な手順については[検索設定](./knowledge-base-creation/upload-documents#3)をご覧ください。
