[ComfyUI](https://www.comfy.org/)は、強力でモジュール化された画像生成プラットフォームです。現在、Difyを利用して、画像のヒントを入力し、生成された画像を取得することが可能です。

## 1. ComfyUIのワークフローが正常に動作することを確認する
[公式ドキュメント](https://docs.comfy.org/get_started/gettingstarted)を参照し、ComfyUIが正しく動作し、画像が生成されることを確認してください。

## 2. ワークフローのAPIファイルをエクスポートする

![](/ja-jp/img/0503c5838f163b54884ca58240281f3f.png)

上の画像のように、ツールバーから「Save(API Format)」を選択します。この選択肢が表示されない場合は、設定で「Dev Mode」を有効にする必要があります。

## 3. DifyにComfyUIを統合する

「ツール > ComfyUI > 認証に移動」でアクセスアドレスを入力します。DifyをDockerでデプロイしている場合、通常このアドレスは「http://host.docker.internal:8188」となります。

## 4. DifyでComfyUIを使用する

「ワークフロー」ツールを開き、「WORKFLOW JSON」にエクスポートしたAPIファイルの内容を入力すると、正常に画像が生成されます。
