# テストドキュメント
これは翻訳機能を検証するためのテストドキュメントです。

---

ドキュメント作成者の方へ:

GitHub Actions は進捗を観測するためにのみ使用します。実際の手順は次のとおりです:

1. ドキュメントを書く
2. push
3. 約 0.5 - 1 分待つ
4. pull request
    1. または、GitHub のウェブページでコミット履歴を確認する

以上です
