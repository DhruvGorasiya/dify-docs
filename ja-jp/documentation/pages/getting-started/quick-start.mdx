## ステップ1：新しいワークフローを作成する（2分）

1. **Studio** > **ワークフロー** > **空白から作成** > **Orchestrate** > **New チャットフロー** > **作成**に移動します

## ステップ2：ワークフローノードを追加する（6分）

<Tip>
  変数を参照したい場合は、まず`{`または`/`を入力すると、ワークフローで利用可能な異なる変数を確認できます。
</Tip>

### 1. 大規模言語モデルノードとアウトプット：質問を理解して回答する

<Info>
  `大規模言語モデル`ノードは、ユーザー入力に基づいてレスポンスを生成するために言語モデルにプロンプトを送信します。APIコール、レート制限、インフラストラクチャの複雑さを抽象化するため、ロジックの設計に集中できます。
</Info>

<Steps>
  <Step title="大規模言語モデルノードを作成">
    `ノード追加`ボタンを使用して大規模言語モデルノードを作成し、スタートノードに接続します
  </Step>
  
  <Step title="モデルを設定">
    デフォルトモデルを選択します
  </Step>
  
  <Step title="システムプロンプトを設定">
    システムプロンプトフィールドに以下を貼り付けます：

    ```text
    The user will ask a question about a country. The question is {{sys.query}}  
    Tasks: 
    1. Identify the country mentioned. 
    2. Rephrase the question clearly. 
    3. Answer the question using general knowledge. 
    
    Respond in the following JSON format: 
    {   
      "country": "<country name>",
      "question": "<rephrased question>",
      "answer": "<direct answer to the question>" 
    }
    ```
  </Step>
  
  <Step title="構造化出力を有効にする">
    **構造化出力を有効にする**により、大規模言語モデルが返すものを簡単に制御し、精密なデータ抽出や条件付きロジックでの下流使用のために、一貫性のある機械可読な出力を保証できます。
    
    - 出力変数構造化をONに切り替える > `設定`をクリックし`JSONからインポート`をクリック
    - 貼り付け：
    
    ```json
    {   
      "country": "string",   
      "question": "string",   
      "answer": "string" 
    }
    ```
  </Step>
  
  <Step title="コードブロック：雑学を取得">

<Info>
  `コード`ノードはコードを使用してカスタムロジックを実行します。視覚的なワークフロー内の必要な場所にコードを正確に挿入でき、バックエンド全体を構築する手間を省きます。
</Info>

<Steps>
  <Step title="コードノードを作成">
    `ノード追加`ボタンを使用して`コード`ノードを作成し、大規模言語モデルブロックに接続します
  </Step>
  
  <Step title="入力変数を設定">
    `入力変数`の名前を「country」に変更し、変数を`structured_output` > `country`に設定します
  </Step>
  
  <Step title="Pythonコードを追加">
    `PYTHON3`に以下のコードを貼り付けます：

    ```python
    def main(country: str) -> dict:     
      country_name = country.lower()     
      fun_facts = {
        "japan": "Japan has more than 5 million vending machines.",
        "france": "France is the most visited country in the world.",
        "italy": "Italy has more UNESCO World Heritage sites than any other country."     
      }     
      fun_fact = fun_facts.get(country_name, f"No fun fact available for {country.title()}.")
      return {"fun_fact": fun_fact}
    ```
  </Step>
  
  <Step title="出力変数名を変更">
    より適切にラベル付けされた変数にするため、出力変数`result`を`fun_fact`に変更します
  </Step>
</Steps>

### 3. 応答ノード：ユーザーへの最終回答

<Info>
  `応答`ノードは返すためのきれいな最終出力を作成します。
</Info>

<Steps>
  <Step title="応答ノードを作成">
    `ノード追加`ボタンを使用して`応答`ノードを作成します
  </Step>
  
  <Step title="応答フィールドを設定">
    応答フィールドに以下を貼り付けます：

    ```text
    Q: {{ structured_output.question }} 

    A: {{ structured_output.answer }} 

    Fun Fact: {{ fun_fact }}
    ```
  </Step>
</Steps>
