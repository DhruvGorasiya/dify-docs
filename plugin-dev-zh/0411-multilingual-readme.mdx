Dify 插件支持多语言 README，将在 Marketplace 等位置根据用户的语言首选项展示对应的 README。

### 放置规范

- /README.md: 必需，英文 README。
- /readme/*: 可选，其他语言 README 文件，格式为`README_<语言标识>.md`，目前支持日本语、葡萄牙语、简体中文。

```bash
...
├── main.py
├── manifest.yaml
├── readme
│   ├── README_ja_JP.md
│   ├── README_pt_BR.md
│   └── README_zh_Hans.md
├── README.md
...
```

### Marketplace 展示规则

[Dify Marketplace](https://marketplace.dify.ai) 界面支持英文、简体中文、日本语等语言，当插件具有对应语言 README 时，将在插件详情页展示用户所使用的语言的 README 文档。
